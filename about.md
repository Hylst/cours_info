# About This Project

## Purpose
This project serves as a personal knowledge base and teaching aid for Docker technologies. It condenses complex documentation into a structured, easy-to-read memo format.

## Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox/Grid layouts, no external frameworks (Vanilla CSS).
- **JavaScript (ES6+)**: Minimal logic for navigation and quiz functionality.
- **Python**: Build script for content modularization.
