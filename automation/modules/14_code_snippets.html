Code Snippets (JS)

Les 20% de code qui font 80% du travail

Pourquoi coder ?
Les nœuds No-Code sont super, mais parfois un nœud "Code" de 3 lignes remplace 15 nœuds "Loop/If/Filter" et tourne 100x plus vite.

A. Arrays (Listes)

.map()

Transformer une liste. (ex: Garder juste les emails).

const users = [ {name: "A", email: "a@test.com"}, {name: "B", email: "b@test.com"} ]; // Résultat: ["a@test.com", "b@test.com"] const emails = users.map(u => u.email);

.filter()

Garder ce qui matche une condition.

const nums = [1, 5, 10, 2]; // Résultat: [5, 10] const big = nums.filter(n => n > 4);

B. Strings (Texte)

.split()

Couper du texte.

const name = "Jean Dupont"; // Résultat: "Jean" const firstname = name.split(" ")[0];

Template Literals

Injecter des variables proprement.

const item = "Pomme"; const price = 10; // "Le prix de Pomme est 10€" const msg = `Le prix de ${item} est ${price}€`;

C. Dates (L'Enfer)

ISO 8601 (Le standard)
2025-02-02T14:00:00.000Z

Utilisez toujours ce format en interne dans n8n.

Format (Luxon)

Dans n8n expressions :

{{ $now.toFormat('dd/MM/yyyy') }}

-> 02/02/2025

D. Regex (Expressions Régulières)

Pour extraire des motifs complexes.

const text = "Contactez-moi au 06 12 34 56 78 merci.";
// Trouve un motif de 10 chiffres (simplifié)
const phone = text.match(/\d{10,}/);
Outil indispensable : Utilisez Regex101.com pour tester vos regex avant de les mettre en prod. L'IA (ChatGPT) est aussi excellente pour générer des Regex.
