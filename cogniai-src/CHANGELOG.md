# Changelog - CogniAI

Toutes les modifications notables apportées à ce projet seront documentées dans ce fichier.

## [1.0.0] - 2025-12-25
### Ajouté
- **Core Engine** : Migration vers React 19 et Vite 6.
- **Système de Diagnostic** : Questionnaire interactif pour évaluer la dépendance à l'IA.
- **Module Socratique** : Galerie de prompts pour encourager la maïeutique avec les LLM.
- **Épreuve de Restitution** : Test interactif de mémorisation et d'encodage (Brain Check).
- **Neurologie & Méthodologie** : Sections détaillées sur le "Cognitive Offloading" et le protocole A.C.T.E.
- **Interface Adaptive** : Design Dark/Glassmorphism optimisé pour desktop et mobile.
- **Optimisation SEO** : Meta tags, Open Graph et structures JSON-LD complets.

### Corrigé
- Ajustement de la réactivité du Navbar pour les petits écrans.
- Correction des délais d'animation sur le Hero Header.

### Sécurité
- Passage de toutes les ressources externes (Google Fonts, Unsplash) en HTTPS pour éviter les avertissements de contenu mixte (Mixed Content).
- Vérification de l'absence de fuite de clés API ou de dépendances backend.

*Version initiale stable.*
