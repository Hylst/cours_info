# À Propos

## L'Auteur
Je suis **Geoffroy Streit**, un passionné de technologies Web, Data, IA et IT. 
Mon objectif est de vulgariser les concepts complexes à travers des interfaces claires, esthétiques et performantes.

## Philosophie du Projet
Ce mémo Java a été conçu avec trois principes en tête :
1.  **Synthèse** : Aller à l'essentiel sans perdre la nuance technique.
2.  **Visuel** : Utiliser des diagrammes et une typographie soignée pour faciliter la mémorisation.
3.  **Performance** : Un site statique ultra-rapide, sans framework lourd, respectueux des ressources.

## Contexte
Ce site fait partie d'un ensemble de ressources éducatives hébergées sur [hylst.fr](https://hylst.fr). Il sert de référence rapide pour les développeurs Java, du débutant à l'expert.
