# À Propos

## Le Projet
Ce Mémo Node.js est né d'un besoin personnel de synthétiser les connaissances backend JavaScript dans un format :
- **Rapide à lire** : Pas de longs blabla, droit au but.
- **Visuel** : Code highlighting, schémas (à venir), design sombre et contrasté.
- **Interactif** : Démos de l'Event Loop directement dans le navigateur.

## Technologie
- **HTML5 / CSS3** (Variables CSS modernes)
- **Vanilla JS** (Pas de framework frontend lourd)
- **Node.js** (Pour le script de build local)
- **Docker / Nginx** (Pour l'hébergement)
