Numpy : Le Socle

Tableaux multidimensionnels & Calcul matriciel

üìñ Comprendre Numpy (Numerical Python)

Les listes Python sont trop lentes pour des millions de calculs. Numpy introduit les Arrays (tableaux).

Un array Numpy force le m√™me type de donn√©es partout (ex: que des `float64`), ce qui permet une optimisation m√©moire extr√™me.

Le Broadcasting permet de faire des op√©rations math√©matiques sur tout le tableau sans boucle `for` explicite. C'est la vectorisation.

Machine Learning

Scikit-Learn (Sklearn)

üìñ Comprendre le Machine Learning avec Sklearn

Le ML consiste √† cr√©er un mod√®le capable d'apprendre des donn√©es pass√©es pour pr√©dire le futur.

Scikit-Learn propose une API unifi√©e g√©niale : 3 √©tapes pour TOUS les mod√®les (R√©gression, Classification, Clustering).

1. Instancier (`model = LinearRegression()`)
2. Entra√Æner (`model.fit(X, y)`)
3. Pr√©dire (`model.predict(new_data)`).

üí° Workflow Type ML : Nettoyage ‚Üí S√©paration Train/Test ‚Üí Choix Mod√®le ‚Üí Entra√Ænement ‚Üí √âvaluation (Score).
Workflow Complet de Classification
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# 1. Pr√©parer Features (X) et Target (y)
X = df.drop(columns=["Survie"])  # Donn√©es
y = df["Survie"]                 # Cible √† pr√©dire

# 2. S√©parer Train (80%) / Test (20%)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 3. Choisir et Entra√Æner le mod√®le
model = RandomForestClassifier(n_estimators=100)
model.fit(X_train, y_train)  # C'est ici que la magie op√®re üß†

# 4. √âvaluer
prediction = model.predict(X_test)
score = accuracy_score(y_test, prediction)

print(f"Pr√©cision du mod√®le : {score*100}%")
