# À propos de Python Web Memo

## Objectif du projet
Ce projet est un **support de cours interactif** et un **mémo technique** dédié au développement Web avec Python. Il a été conçu pour offrir une vision panoramique et structurée des technologies modernes (Flask, FastAPI, Django) tout en approfondissant les concepts clés du backend.

## Philosophie
- **Accessibilité** : Un format "mémo" condensé pour retrouver rapidement une syntaxe ou un concept
- **Pédagogie** : Utilisation de quiz, de schémas Mermaid et d'exemples de code "prêts à l'emploi"
- **Performance** : Un site statique léger, rapide et ultra-portable (déployable via Docker en quelques secondes)
- **Esthétique** : Un design système sombre (Dark Mode) optimisé pour le confort de lecture des développeurs
- **Responsive** : Navigation adaptative avec menu hamburger pour une expérience optimale sur tous supports

## Public Cible
- **Étudiants** en informatique découvrant le développement web
- **Développeurs Python** souhaitant explorer l'écosystème web
- **Formateurs** cherchant un support de rappel pour leurs cours
- **Autodidactes** en quête d'une ressource complète et structurée

## Structure du contenu
Le contenu est organisé de manière progressive :
1. **Panorama** : Comprendre pourquoi et comment Python est utilisé pour le Web
2. **Setup** : Maîtriser l'environnement de développement professionnel
3. **Frameworks** : Comparaison et mise en pratique (Flask, FastAPI, Django)
4. **Data & Sécurité** : L'essentiel sur les ORM et la protection des données
5. **Concepts Avancés** : Asynchronisme, WebSockets et tâches de fond
6. **DevOps** : Tests, Qualité et Déploiement (Docker)

## Fonctionnalités Clés
- **14 modules thématiques** couvrant tout l'écosystème Python Web
- **Navigation responsive** avec menu hamburger sur mobile
- **Quiz interactif** pour valider ses acquis
- **Copie de code en un clic** sur tous les exemples
- **Diagrammes visuels** (Mermaid.js) pour comprendre les architectures
- **Design system moderne** avec variables CSS personnalisables
- **SEO optimisé** pour une meilleure visibilité
