# À Propos

## Le Projet
Ce mémo React.js s'inscrit dans un projet personnel plus large de création de ressources pédagogiques pour développeurs (Web, Data, IA). Il vise à fournir une référence rapide, visuelle et interactive pour les concepts clés de React, souvent difficiles à visualiser pour les débutants.

L'objectif est double :
1. **Consolidation** : Aider l'auteur à ancrer ses connaissances en les synthétisant.
2. **Partage** : Offrir une ressource gratuite et accessible à la communauté francophone.

Ce projet est hébergé sur un VPS (Hostinger) géré via Coolify, tournant sur des conteneurs Docker (Alpine/Nginx) pour une performance et une sécurité optimales.
